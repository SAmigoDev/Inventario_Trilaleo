# Inventario_Trilaleo
Sistema de inventario para administración y control de almacén.

Bienvenidos a la demostración de nuestro nuevo Sistema de Gestión e Inventario para Importadora Trilaleo. Como Pyme con más de 25 años, esta herramienta marca un hito en nuestra misión de modernizar procesos y dejar atrás la fuerte dependencia del papel, buscando una gestión eficiente, intuitiva y, sobre todo, escalable.
