Requisitos Previos Necesarios en el PC
Antes de iniciar la instalación, asegúrese de que el usuario cumpla con los siguientes requisitos en su computadora:

1. Sistema Operativo
Sistema Operativo: Windows 10 o Windows 11 (se recomienda la versión de 64 bits).

Permisos: El usuario debe tener permisos de administrador para ejecutar el instalador y realizar la instalación en las carpetas del sistema.

2. Instalación de Python (Obligatorio)
Es crucial tener el intérprete de Python instalado en el equipo. Si no conoce la versión exacta requerida por el desarrollador del programa, se recomienda usar una versión moderna (Python 3.8 o superior).

Pasos Clave para la Instalación de Python:

Descargue el instalador de Python desde el sitio web oficial.

Al ejecutar el instalador, es fundamental marcar la casilla "Add Python XX.X to PATH" (Agregar Python XX.X a la variable PATH) en la primera pantalla. Esto permite que el sistema operativo localice el intérprete de Python, lo cual es vital si el ejecutable depende de él. 3. Seleccione "Install Now" (Instalar ahora) y siga los pasos.

3. Requisitos de Hardware (Mínimos Recomendados)
Procesador: Procesador de doble núcleo (Dual-Core) o superior.

RAM: 4 GB de memoria RAM o más.

Almacenamiento: Espacio disponible en disco duro (generalmente unos pocos GB, más el espacio para Python).

Guía de Instalación de Importadora_Trilaleo.exe
A continuación, se detallan los pasos estándar para la instalación de un programa ejecutable en Windows:

Paso 1: Localizar el Archivo
Asegúrese de que el archivo Importadora_Trilaleo.exe se haya descargado completamente.

Paso 2: Ejecutar el Instalador
Haga doble clic sobre el archivo Importadora_Trilaleo.exe.

Si Windows muestra una advertencia de Control de cuentas de usuario (UAC), haga clic en "Sí" para permitir que el instalador realice cambios en el equipo.

Paso 3: Seguir el Asistente de Instalación
Pantalla de Bienvenida: Revise la información y haga clic en "Next" (Siguiente).

Acuerdo de Licencia: Lea el acuerdo de licencia (EULA). Si está de acuerdo, seleccione la opción "I accept the terms in the License Agreement" (Acepto los términos del Acuerdo de Licencia) y haga clic en "Next" (Siguiente).

Ruta de Instalación: Se le pedirá seleccionar la carpeta donde se instalará el programa.

Deje la ruta predeterminada (generalmente C:\Program Files\Importadora_Trilaleo) a menos que tenga una razón específica para cambiarla.

Haga clic en "Next" (Siguiente).

Confirmación y Comienzo: Revise la configuración elegida y haga clic en "Install" (Instalar) para iniciar la copia de archivos.

Paso 4: Finalizar la Instalación
Espere a que la barra de progreso se complete.

Una vez finalizado, el asistente mostrará un mensaje de confirmación, como "Installation Complete" (Instalación completada).

Haga clic en "Finish" (Finalizar) para cerrar el instalador.

El programa ya debería estar instalado y un acceso directo aparecerá en el Escritorio y/o en el menú de Inicio.
